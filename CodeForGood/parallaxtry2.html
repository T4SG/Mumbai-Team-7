CSS3 Parallax by Martin Ivanov (@wemakesitesnet). No JavaScript. No Hassle.

You gonna love this one for it's simple HTML, elegant CSS and lack of any JavaScript. Scroll the page down to see how it's done. And it's responsive too! Download here and play with the code! Note that as of September 2014, Internet Explorer still does not fully support CSS 3D transforms, so for that browser the effect is impaired (but the layout is okay).

On Chrome, due to a longtime un-addressed issue, the effect will not be applied on mouse wheel scroll, but if you drag the scrollbar it will kick in. Fortunately, there a lot of JavaScript solutions for smooth scrolling.

CSS3 Rule #1

Start coding your beautiful CSS with the assumption that you might not need any JavaScript at all. And you won't!
